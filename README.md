* Kodevio User's App *

## Features

* Fetch users from REST API : https://jsonplaceholder.typicode.com/users
* Display user list With :
   - Name
   - Email
   - Company Name
* User details screen showing :
    - Name
    - Username
    - Email
    - Phone
    - Website
    - Address

* Loading indicator while fetching data
* Error handling for failed API requests
* Pull- to - refresh
* Search users by Name
* Clean architecture with separation of concerns
